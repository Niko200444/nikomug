Soft Premium Vintage Newspaper Theme

Bu versiyada funksionallığa toxunulmayıb. Əsas dəyişiklik styles.css-in sonunda əlavə edilmiş “SOFT PREMIUM NEWSPAPER REFINEMENT” override blokudur.

Dizayn istiqaməti:
- əvvəlki sərt qara/ağ kontrast yumşaldıldı;
- incə editorial grid və kağız teksturası əlavə edildi;
- double/thick border-lər daha real, nazik hairline border-lərlə əvəz edildi;
- kartlar, düymələr, statistik panel və modal premium, oxunaqlı newspaper UI formasına salındı;
- dark mode də daha yumşaq evening-edition estetikasına uyğunlaşdırıldı;
- app.js, db.js və əsas struktur funksional olaraq dəyişdirilmədi.
